# 职场小白第一课

围绕真实职场场景进行练习，快速掌握结论先行、并列分点、证据支撑与结构化表达，提升周报、纪要、方案、复盘等常见输出的质量。

## 功能
- 案例训练：围绕职场典型场景进行练习，填写“结论 + 3–5 分点 + 证据”
- 智能评分：从主旨开篇、并列分点、证据支撑、故事线、结构步骤五维度打分（总分 100）
- 进度保存：草稿与评分保存在浏览器 `localStorage`
- 知识模块：写作原则、表达结构、故事线（SCQA）、常见场景速览
- 主题自适应：按时间与系统偏好自动浅/深色，也可手动切换

## 本地运行
- 启动：在 `pyramid-edu` 目录运行 `python -m http.server 8001`
- 访问：`http://localhost:8001/`
- 使用：选择案例 → 填写输出 → 评分与提纲 → 保存草稿

## 修改案例
- 编辑 `data/cases.json`，保持 JSON 语法正确
- 字段示例：
```json
{
  "id": "email-update",
  "title": "项目周报邮件",
  "scene": "职场汇报",
  "level": "入门",
  "goal": "用结构化方式写一封清晰的项目周报",
  "brief": "收件人包括主管与跨部门伙伴，要求一屏内读完核心信息并可展开细节。",
  "context": "背景与提示…"
}
```


## 常见问题
- 无法评分：确保填写了“结论 + 分点 + 证据”，点击“评分与提纲”
- JSON 语法错误：检查逗号与引号，建议使用 JSON 校验
- 主题太暗或太亮：点击页面顶部“主题”按钮手动切换
