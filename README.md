# 金字塔教育程序（Pyramid Education Trainer）

通过实际案例逐步训练用户，让其熟练运用金字塔原理（结论先行、以上统下、归类分组、逻辑递进）到工作、生活与学习场景。

## 功能
- 案例训练：填写“结论 + 3–5 分点 + 证据”，一键生成评分与提纲
- 评分维度：主旨开篇、MECE 分点、证据支撑、SCQA 故事线、结构步骤（总分 100）
- 进度保存：草稿与评分保存到浏览器 `localStorage`
- 知识模块：核心原则、结构技巧、SCQA、常见场景速览
- 主题自适应：按时间与系统偏好自动浅/深色，支持手动切换

## 目录
```
pyramid-edu/
  index.html
  styles.css
  app.js
  data/
    cases.json    # 训练案例
```

## 本地运行
- 启动：在 `pyramid-edu` 目录运行 `python -m http.server 8001`
- 访问：`http://localhost:8001/`
- 使用：案例训练 → 填写输出 → 评分与提纲 → 保存草稿

## 修改案例
- 编辑 `data/cases.json`，保持 JSON 语法正确
- 字段示例：
```json
{
  "id": "email-update",
  "title": "项目周报邮件",
  "scene": "职场汇报",
  "level": "入门",
  "goal": "用金字塔结构写一封清晰的项目周报",
  "brief": "收件人包括主管与跨部门伙伴，要求一屏内读完核心信息并可展开细节。",
  "context": "背景与提示…"
}
```


## 常见问题
- 无法评分：确保填写了“结论 + 分点 + 证据”，尝试点击“评分与提纲”
- JSON 语法错误：检查逗号与引号，建议使用 JSON 校验
- 主题太暗或太亮：点击页面顶部“主题”按钮手动切换
